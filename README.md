# Road-to-Heaven-Zhongnan-Mountain-Hermit-Map
This project visualizes the hermit sites visited by Bill Porter (Red Pine) in his book **"Road to Heaven: Encounters with Chinese Hermits"** (《空谷幽兰》).

# 🗺️ Features
- Real administrative boundaries** from Alibaba DataV API
- 44 documented locations** from the book and supplementary sources
- 5 location types** with distinct color coding:
  - 🏯 Buddhist Temples (22)
  - 🛕 Taoist Temples (11)
  - 🏔️ Mountain Peaks (6)
  - 🏚️ Hermit Huts (3)
  - 📍 Historical Sites (2)
- Smart label placement using `ggrepel` to avoid overlaps
- High-resolution output (300+ DPI)
- Chinese font support for proper rendering

# Prerequisites
Make sure you have R (4.0+) installed on your system.
